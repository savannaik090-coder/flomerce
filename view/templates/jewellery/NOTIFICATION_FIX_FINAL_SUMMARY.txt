TECHNICAL DETAILS:
================================================================================

THE CRITICAL FIX (Why notifications now work):

BEFORE (Wrong):
  messaging.sendMulticast({
    tokens: batch,
    notification: { title, body },     // Top-level ← IGNORED for web!
    webpushConfig: {
      notification: { title, body }    // Duplicate ← CONFLICTS
    }
  })

AFTER (Correct):
  messaging.sendMulticast({
    tokens: batch,
    webpushConfig: {                   // Web push ONLY
      notification: { 
        title, body, 
        icon, badge, 
        clickAction,                   // NEW: Enables navigation
        tag                            // NEW: Prevents duplicates
      },
      data: { link, category },        // Custom data
      fcmOptions: { link }             // NEW: FCM navigation config
    }
  })

This matches Firebase Cloud Messaging spec for web browsers.
